# Shedge Stores – E-Commerce Admin Panel

A full-stack MERN Product Management System for managing products in an e-commerce admin panel.

## Live Links

- Frontend Live URL: https://shedge-stores-frontend.vercel.app
- Backend Live URL: https://shedge-stores-backend.onrender.com
- Frontend GitHub Repository: https://github.com/Omkar-codes8055/shedge-stores-frontend
- Backend GitHub Repository: https://github.com/Omkar-codes8055/shedge-stores-backend

## Project Features

- Admin Register and Login
- JWT Authentication
- Add Product with Image Upload
- View All Products
- Edit Product Details
- Delete Product
- Product Details Page
- Search Products by Name
- Filter Products by Category and Status
- Sort Products by Name, Price and Date
- Pagination
- Dashboard Statistics
- Dynamic Categories
- Toast Notifications
- Responsive Admin Panel UI

## Technology Stack

### Frontend

- React.js
- Vite
- React Router DOM
- Tailwind CSS
- Axios
- React Hot Toast
- React Icons
- SweetAlert2

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- BcryptJS
- CORS
- Dotenv
